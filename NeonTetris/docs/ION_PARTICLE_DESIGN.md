# NeonTetris 离子粒子视觉优化 v2

## 优化目标

从"物理感强"升级到"轻、飘逸、虚化"的离子效果

**视觉目标**：
- 单个离子由三部分组成：核心、壳、外层
- 外层虚化、半透，有朦胧感
- 运动路径大方向符合物理，局部随机扰动
- 整体感觉轻、飘逸、如同霓虹离子在空中飘动

---

## 核心改进

### 1. 三层离子结构 ✅

每个粒子渲染为三层同心圆，从外到内：

#### 第一层：外层虚化（最大，最虚）
- **大小**：2.5x 核心大小
- **透明度**：10%（极虚）
- **作用**：朦胧的外晕，给人虚化感
- **效果**：像离子的能量场扩散

#### 第二层：中层壳（中等，半透）
- **大小**：1.6x 核心大小
- **透明度**：35%（半透）
- **作用**：离子的壳层，半透感
- **效果**：像离子的外壳结构

#### 第三层：核心（最小，最亮）
- **大小**：1.0x（原始大小）
- **透明度**：100%（完全不透）
- **作用**：离子的核心，最亮
- **效果**：像离子的能量核心

```
    外层虚化（10%）
        ↓
    中层壳（35%）
        ↓
    核心亮（100%）
```

**视觉效果**：从外到内逐渐变亮，形成立体的离子感

### 2. 运动路径优化 ✅

#### 大方向：符合物理
- **重力**：根据粒子类型调整，lockFlash 极轻（0.02x），firework 向上飘（-0.25x）
- **风场**：全局风力影响所有粒子
- **摩擦**：减速粒子运动

#### 局部：随机扰动
- **Perlin 噪声**：加快变化速度（3.0x，原来 2.0x）
- **湍流强度**：增加局部随机性（120x，原来 80x）
- **噪声采样频率**：提高（0.015，原来 0.01）

**效果**：粒子大方向下落/上升/飘动，但局部轨迹蜿蜒、随机、自然

## 渲染流程

```swift
for particle in particles {
    let progress = particle.lifeProgress  // 0-1
    let size = particle.size * progress   // 随生命周期缩小
    
    // 第一层：外层虚化（2.5x，10%）
    drawCircle(center: particle.position, radius: size * 2.5, 
               color: particle.color, opacity: progress * 0.08)
    
    // 第二层：中层壳（1.6x，35%）
    drawCircle(center: particle.position, radius: size * 1.6, 
               color: particle.color, opacity: progress * 0.35)
    
    // 第三层：核心（1.0x，100%）
    drawCircle(center: particle.position, radius: size, 
               color: particle.color, opacity: progress)
}
```

## 物理模拟流程

```
每帧（0.016s）：
1. 应用全局风场（0.08x）
2. 应用重力（根据粒子类型，0.02-0.4x）
3. 应用 Perlin 噪声（3.0x 速度，120x 强度）
4. 更新速度 += 加速度
5. 应用摩擦 *= friction（0.88-0.99）
6. 更新位置 += 速度
7. 重置加速度
```
