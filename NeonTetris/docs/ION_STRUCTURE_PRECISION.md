# NeonTetris 离子粒子结构优化 v4

## 优化目标

从"任意比例"升级到"精确比例"的离子结构

**视觉目标**：
- 核心：半径占整个离子圆的 1/4，透明度 80%
- 壳层：半径占整个离子圆的 1/4，透明度 50%
- 外层：半径占整个离子圆的 2/4（一半），透明度 20%
- 整体感觉：从内到外逐渐虚化，形成清晰的离子层次感

## 核心改进

### 1. 精确的三层比例 ✅

#### 整个离子圆的结构

假设整个离子圆的半径为 `R`：

```
    ┌─────────────────────────────┐
    │                             │
    │      外层（2/4 = 1/2）      │  透明度 20%
    │    ┌─────────────────────┐  │
    │    │                     │  │
    │    │  壳层（1/4）        │  │  透明度 50%
    │    │  ┌───────────────┐  │  │
    │    │  │               │  │  │
    │    │  │ 核心（1/4）   │  │  │  透明度 80%
    │    │  │               │  │  │
    │    │  └───────────────┘  │  │
    │    │                     │  │
    │    └─────────────────────┘  │
    │                             │
    └─────────────────────────────┘
```

#### 具体尺寸

假设 `size` 是粒子的基础大小（直径）：

| 层级 | 占比 | 直径 | 半径 | 透明度 |
|------|------|------|------|--------|
| 外层 | 2/4 | size × 2.0 | size × 1.0 | 20% |
| 壳层 | 1/4 | size × 1.0 | size × 0.5 | 50% |
| 核心 | 1/4 | size × 0.5 | size × 0.25 | 80% |

**说明**：
- 外层直径 = size × 2.0（因为外层占 2/4，所以直径是 size 的 2 倍）
- 壳层直径 = size × 1.0（因为壳层占 1/4，相对于核心是 1.0 倍）
- 核心直径 = size × 0.5（因为核心占 1/4，所以直径是 size 的 0.5 倍）

### 2. 透明度设置 ✅

| 层级 | 透明度 | 说明 |
|------|--------|------|
| 外层 | 20% | 最虚，几乎看不见 |
| 壳层 | 50% | 半透，能看见但不清晰 |
| 核心 | 80% | 最亮，清晰可见 |

**效果**：从外到内逐渐变亮，形成清晰的离子层次感

### 3. 渲染顺序 ✅

从外到内渲染，确保后渲染的层覆盖前面的层：

```
1. 绘制外层（最大，最虚）
   ↓
2. 绘制壳层（中等，半透）
   ↓
3. 绘制核心（最小，最亮）
```

**效果**：后渲染的层会覆盖前面的层，形成正确的视觉效果

## 代码实现

### 渲染代码

```swift
for particle in particles.particles {
    let progress = particle.lifeProgress
    let size = particle.size * CGFloat(progress)
    
    // 第一层：外层（2/4，20%）
    let outerSize = size * 2.0
    let outerRect = CGRect(x: particle.position.x - outerSize / 2,
                          y: particle.position.y - outerSize / 2,
                          width: outerSize,
                          height: outerSize)
    context.fill(
        Path(ellipseIn: outerRect),
        with: .color(particle.outerColor.opacity(Double(progress) * 0.20))
    )
    
    // 第二层：壳层（1/4，50%）
    let shellSize = size * 1.0
    let shellRect = CGRect(x: particle.position.x - shellSize / 2,
                          y: particle.position.y - shellSize / 2,
                          width: shellSize,
                          height: shellSize)
    context.fill(
        Path(ellipseIn: shellRect),
        with: .color(particle.shellColor.opacity(Double(progress) * 0.50))
    )
    
    // 第三层：核心（1/4，80%）
    let coreSize = size * 0.5
    let coreRect = CGRect(x: particle.position.x - coreSize / 2,
                         y: particle.position.y - coreSize / 2,
                         width: coreSize,
                         height: coreSize)
    context.fill(
        Path(ellipseIn: coreRect),
        with: .color(particle.color.opacity(Double(progress) * 0.80))
    )
}
```
