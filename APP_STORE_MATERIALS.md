## 4. 应用截图规格

### macOS App Store 截图要求
- **最小尺寸**: 1280 x 720 像素
- **推荐尺寸**: 1920 x 1080 像素 或 2560 x 1440 像素
- **格式**: PNG 或 JPEG
- **文件大小**: 每张不超过 10MB

### 建议截图内容

| 截图 | 内容 | 尺寸建议 |
|------|------|----------|
| 1 | 主游戏界面 - 展示游戏板和霓虹效果 | 1920x1080 |
| 2 | 游戏进行中 - 消行时的粒子效果 | 1920x1080 |
| 3 | 排行榜界面 - 展示最高分 | 1920x1080 |
| 4 | 主题选择 - 展示8种主题 | 1920x1080 |
| 5 | 设置界面 - 音效/速度设置 | 1920x1080 |

### 截图技巧
1. 使用 Command + Shift + 4 截取游戏窗口
2. 确保游戏界面清晰展示
3. 可以添加边框或文字说明（可选）
