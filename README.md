# NeonTetris
NeonTetris 是一个功能完整的 macOS 俄罗斯方块游戏，具有华丽的七彩离子粒子效果、完整的音效系统、自定义主题、排行榜和烟花庆祝效果

NeonTetris is a fully-featured Tetris game for macOS, complete with gorgeous colorful ionic particle effects, a full-fledged audio system, custom themes, a leaderboard, and celebratory firework effects.
